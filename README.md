# 💼 Software Implementation Engineer Portfolio

A modern, responsive, and professional portfolio website showcasing the journey, skills, and achievements of a **Software Implementation Engineer**. This portfolio highlights expertise in software implementation, client onboarding, system configuration, project coordination, and successful go-live execution.

---

## 👩‍💻 About

This portfolio represents a detail-oriented Software Implementation Engineer with strong analytical, problem-solving, and communication skills. It demonstrates experience in delivering seamless software implementations, coordinating end-to-end projects, configuring systems according to client requirements, and ensuring successful client onboarding and adoption.

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/prashantnadar2210/mayuri-thackrey-portfolio.git
```

### Navigate to the Project Directory

```bash
cd mayuri-thackrey-portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
